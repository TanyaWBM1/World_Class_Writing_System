# PROJECT_BOOTSTRAP

## Purpose

This document marks the initial bootstrap state for the World Class Writing System.

The repository is intentionally scaffold-only in this phase. It establishes the architecture, ownership boundaries, and future file locations defined by `PROJECT_EXECUTION_ORDER.md`.

## Bootstrap Scope

This pass includes:
- source-of-truth folder structure
- starter system charter files
- base subagent definition files
- placeholder schemas
- placeholder library directories
- stress-test layer scaffolding
- innovation workspace

This pass excludes:
- runtime generation implementation
- enforcement implementation
- validation implementation
- benchmark fixture implementation
- research prompt implementation
- migration of legacy repo logic

## Build Order Locked For This Repo

1. Skeleton
2. Charter
3. Base agents
4. Schemas
5. Library folders
6. Stress-test layer
7. Research prompt pack
8. Research ingestion system
9. Library architecture

## Governance

- This is a new repository skeleton, not a copy of the old writing repo.
- Future runtime changes must respect the system charter and schema contracts.
- Research should flow through `research/raw_exports/` to structured findings and proposals before becoming library assets.
