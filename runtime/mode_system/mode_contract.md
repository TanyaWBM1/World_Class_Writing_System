# Mode Contract

`creative_writing` and `acf_lite` are mutually exclusive runtime operating modes.

Creative Writing:

- imagination-first
- ACF disabled
- symbolic reasoning allowed
- truth-validation requirements not forced
- Tanya voice, grit, lexical grounding, phrase diversity, and pattern controls remain active

ACF Lite:

- reality-first
- ACF requirements active
- imagination-heavy generation disabled
- symbolic reasoning disabled
- strict claim framing required
- disconfirmation condition required
- external collider required
- uncertainty disclosure required
- outcome window required

Global rule:

- mode blending is not allowed
- both modes cannot be active at the same time
